# Custom Recipes

> Adds crafting, smelting, smoking, campfire, and stonecutting recipes missing from vanilla.

## Chainmail Armor

Chainmail armor can now be crafted using **Iron Bars** in a crafting table. Previously obtainable only via trading or mob drops.

**Helmet** — shaped (`AAA` / `A A`)

```
[A][A][A]
[A][ ][A]
```

**Chestplate** — shaped (`A A` / `AAA` / `AAA`)

```
[A][ ][A]
[A][A][A]
[A][A][A]
```

**Leggings** — shaped (`AAA` / `A A` / `A A`)

```
[A][A][A]
[A][ ][A]
[A][ ][A]
```

**Boots** — shaped (`A A` / `A A`)

```
[A][ ][A]
[A][ ][A]
```

`A` = Iron Bars

## Diamond Recycle

Any damaged or unwanted diamond gear can be smelted in a **Blast Furnace** to recover 1 Diamond.

| Input                                                                                                             | Output  | XP  | Time           |
|-------------------------------------------------------------------------------------------------------------------|---------|-----|----------------|
| Diamond Axe, Boots, Chestplate, Helmet, Hoe, Horse Armor, Leggings, Nautilus Armor, Pickaxe, Shovel, Spear, Sword | Diamond | 1.0 | 5s (100 ticks) |

---

## Painting Selection

Every painting variant can be selected individually via the **Stonecutter**. Place any `Painting` item in the stonecutter to choose a specific variant as output (1 → 1).

---

## Rotten Flesh → Leather

Rotten Flesh can be cooked into **Leather** in three ways:

| Station  | Input        | Output  | XP   | Time            |
|----------|--------------|---------|------|-----------------|
| Furnace  | Rotten Flesh | Leather | 0.1  | 10s (200 ticks) |
| Smoker   | Rotten Flesh | Leather | 0.1  | 5s (100 ticks)  |
| Campfire | Rotten Flesh | Leather | 0.05 | 30s (600 ticks) |

---

## Wood ↔ Log Conversion
