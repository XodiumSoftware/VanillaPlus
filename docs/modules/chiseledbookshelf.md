# Chiseled Bookshelf

Displays information on the action bar when a player left-clicks a **chiseled bookshelf** from the front face.

## Config

| Key       | Type      | Default | Description        |
|-----------|-----------|---------|--------------------|
| `Enabled` | `Boolean` | `false` | Enable this module |

## Behaviour

- Left-clicking the front face of a chiseled bookshelf shows book and enchantment info on the action bar.
- No commands or permissions required.
