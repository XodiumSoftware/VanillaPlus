# Scoreboard

Adds a `/leaderboard` command, which toggles a persistent scoreboard overlay for the player.

## Commands & Permissions

| Command        | Aliases       | Permission                | Default  | Description                              |
|----------------|---------------|---------------------------|----------|------------------------------------------|
| `/leaderboard` | `lb`, `board` | `vanillaplus.leaderboard` | Everyone | Toggle the leaderboard scoreboard on/off |

## Behaviour

- The scoreboard is applied automatically to players on `join`.
- Players can toggle it off and on with `/leaderboard`.
