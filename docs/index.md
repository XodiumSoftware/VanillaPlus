# VanillaPlus

VanillaPlus is a [Paper](https://papermc.io/) Minecraft plugin written in Kotlin that enhances the vanilla gameplay without replacing it. Every feature is modular and independently toggleable via a single `config.json`.

## Features at a glance

| Category         | What it adds                                                     |
|------------------|------------------------------------------------------------------|
| **World**        | Anti-grief per-mob, double doors, nether portal control          |
| **Player**       | Nicknames, skull drops, XP bottles, sitting, pet transfer        |
| **Chat**         | Custom format, whispers, welcome text                            |
| **Inventory**    | Search nearby containers, unload to nearby containers            |
| **Enchantments** | 6 custom + 2 vanilla extensions                                  |
| **Cosmetics**    | Custom MOTD, tab list, locator bar colour, Xaero map integration |

## Requirements

- **Paper** 1.21.11+
- **Java** 21+
